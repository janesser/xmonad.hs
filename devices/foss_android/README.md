# Enforcing FOSS on android devices

What triggered me in first place was [LibreFind](https://github.com/jksalcedo/librefind), a promising alternative finder, which is actually being populated.

## e/OS (based on lineageos)

Key points (to me):

- Supported [e/OS devices](https://doc.e.foundation/devices).
- FOSS can't be for free. [e/OS funding](https://e.foundation/de/donate-2/) options available.
- [Parental control](https://doc.e.foundation/support-topics/parental-control) available.

## Awesome FOSS Alternatives

| Preinstalled | Alternative | Comment |
| ------------ | ----------- | ------- |
| Play Store | [Droidify](https://droidify.app/) | F-Droid Client |
| Camera | [Open Camera](https://opencamera.org.uk/) | Merely feature complete. i wonder if the native application has any optimizations i should be missing.|
| Browser | [Web Libre](https://github.com/FaFre/WebLibre) | Decent privacy-focus, nice embedding with links coming in from other apps |
| Authenticator | [Ente Auth](https://ente.com/auth/) | Migration was painless. Still need to setup backup. |
| One Drive | [Nextcloud](https://github.com/nextcloud/android) | Flawless so far, works across all machines. I consider migrating mail/contacts/agenda there. |
| Agenda | [Fossily](https://www.fossify.org/apps/) Calendar & [DAVx5](https://www.davx5.com/) | Seldomly need to oversee DAVx5 priviledges, which might be lost. |
| Contacts | [Fossily](https://www.fossify.org/apps/) Contacts & [DAVx5](https://www.davx5.com/) | Seldomly need to oversee DAVx5 priviledges, which might be lost. |
| Gmail | [Fairmail](https://email.faircode.eu/) | "_Stop crying. Stop using closed-source apps. Use Fairmail_" |
| Gallery | [Lavender Photos](https://github.com/kaii-lb/LavenderPhotos) | I have QuMagic running to sync with my qnap-nas. |
| Youtube Music | [RiPlay](https://github.com/fast4x/RiPlay) | Needed non stop playlists for car-driving. |
| Gboard | [Any Soft Keyboard](https://github.com/AnySoftKeyboard/AnySoftKeyboard) | I am learning 'neo2' layout now. Swipe is beta. |
